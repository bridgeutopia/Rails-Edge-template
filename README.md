Rails Edge Template
============================

Gems included
-----------------------------

* Devise (for authentication) 
* Formtastic
* RSpec
* Cucumber
* Friendly ID
* Kaminari (for pagination) 
* Twitter Bootstrap 
* Guard and Spork 
                                                                                                         
Using The Template
-----------------------------

Run this: 

    rails new appname -m https://raw.github.com/bridgeutopia/rails-edge-template/master/template.rb
